# Agricultural Surveillance Drone

## Project Overview
This project presents the design and development of an Agricultural Surveillance Drone aimed at improving modern farming practices through aerial monitoring, automation, and real-time data collection.

## Objectives
- Real-time crop and field monitoring
- GPS-based autonomous navigation
- Farm security and intrusion detection
- Support for precision and sustainable agriculture

## Technologies Used
- Quadcopter Drone Platform
- Flight Controller (APM / Pixhawk)
- GPS Module
- FPV Camera System
- Python-based Simulation
- Telemetry and RC Control

## Key Features
- Real-time video surveillance
- Autonomous waypoint navigation
- Crop health monitoring
- Improved irrigation management
- Expandable for AI and IoT integration

## Applications
- Precision agriculture
- Crop health assessment
- Farm security
- Environmental monitoring
